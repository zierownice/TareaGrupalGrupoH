## Descripción del Proyecto

EcoAcciónRD es una aplicación web diseñada para medir y celebrar el impacto de acciones ecológicas en República Dominicana. La plataforma permite a grupos estudiantiles y organizaciones comunitarias registrar sus actividades ambientales en tres categorías principales:

- **Reforestación:** Seguimiento de árboles plantados y su impacto en la captura de CO₂
- **Reciclaje:** Registro de materiales reciclados y reducción de emisiones
- **Ahorro Energético:** Monitoreo de energía ahorrada y su contribución ambiental

La aplicación calcula automáticamente el impacto estimado en reducción de CO₂, proporcionando visualizaciones claras del impacto colectivo de todas las acciones registradas.
